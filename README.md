# cost-gateway

## usage

```
# generate server code
make generate

# db migration
make migrate-version
make migrate-up-all
make migrate-up-one
make migrate-down-one
make migrate-force-v v={version}
make migrate-create name="{migration name}"
make migrate options="{options}"
```
